

This project, **AlgoPlay Quest (also known as the DSA Game Arena), is a premium, interactive web application designed to help users learn and practice Data Structures and Algorithms (DSA) through engaging gameplay rather than just static theory.

Core Features & Games
The platform currently features several interactive "quests" that turn complex algorithmic concepts into playable experiences:
*   Binary Search Quest: An interactive game to master one of the most fundamental search algorithms.
*   Shortest Path Quest: A graph-based challenge where you visualize and solve pathfinding problems (likely using Dijkstra's or BFS logic).
*   Tower of Hanoi Quest: A classic puzzle game that helps build a strong intuition for recursive thinking.
*   Interactive Dashboard: A center to track your progress and stats across different algorithmic challenges.

### 🛠️ Technology Stack**
It is built using a modern, high-performance tech stack:
*   Frontend: Built with React 18 and Vite for a fast, responsive user interface.
*   Design: Uses Tailwind CSS for a sleek "glassmorphism" aesthetic and **Lucide React** for premium iconography.
*   Animations: Framer Motion** powers the smooth transitions and micro-interactions that make the games feel alive.
*   UI Components: Uses Radix UI primitives for accessible, high-quality components (like dialogs, menus, and progress bars).
*   Data Handling: Powered by TanStack Query (React Query) for efficient state management and **Zod** for robust data validation.

What Makes it Special?
Unlike traditional coding platforms, this project focuses on **visual learning** and **gamification**. It uses a dark-themed, premium design to create an immersive environment that makes studying DSA feel like playing a strategy game.
